## ✨ Recursos do Projeto

- **Responsividade Total:** Interface adaptável para telas de celulares, tablets e desktops.
- **Design Moderno:** Fundo com gradiente em tons de roxo e azul, formulário centralizado e campos estilo *underline*.
- **Campos Completos:** Nome, Email, Telefone, Sexo (Radio Buttons), Data de Nascimento, Cidade, Estado e Endereço.
- **Feedback Visual:** Efeitos de transição nos campos e botão interativo com efeito hover.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica do formulário.
- **CSS3:** Estilização, layout flexível e regras de responsividade (Media Queries).
